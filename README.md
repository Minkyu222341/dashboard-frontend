# Dashboard Frontend

Next.js와 TypeScript 기반의 대시보드 프론트엔드 프로젝트입니다.

## 기술 스택

- Next.js 14.2.3
- TypeScript
- React 18
- TailwindCSS
- ESLint

## 기능

- 사이트 상태 대시보드 UI
- 상태 카드 컴포넌트
- 상태 차트 시각화
- 사이트 상태 테이블

## 시작하기

```bash
# 패키지 설치
npm install
```

```bash
# 개발 서버 실행 (http://localhost:3000)
npm run dev
```
